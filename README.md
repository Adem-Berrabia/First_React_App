# React-App

First Website , App build with React JS.

## Built with

- [React-JS](https://react.dev/) library

## Run Locally

Clone the project

```bash
  git clone https://github.com/ITSADEM/First_React_App
```

Go to the project directory

```bash
  cd First_React_App
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm start
```
